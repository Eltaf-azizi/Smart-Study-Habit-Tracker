<h1 align="center">Smart Study Habit Tracker</h1>
  
Smart Study & Habit Tracker is a desktop application designed to help students improve their study consistency, build healthy habits, manage tasks, and track progress over time. The app focuses on clarity, motivation, and long-term growth rather than short-term pressure.

It is fully functional as a standalone desktop app and structured to be expanded into a high-level product in the future.

## Why This Project Exists

Many students struggle not because they lack ability, but because they lack visibility into how they use their time. Existing tools are either too simple to be useful or too complex to maintain consistently.

This project solves that by providing:

 - Clear study time tracking
 - Simple habit building
 - Task organization
 - Meaningful daily and weekly insight
 - Optional social motivation through leaderboards

All without invading privacy or overcomplicating the experience.


## Key Features

### Study Timer

 - Track focused study sessions by subject
 - Start and stop sessions manually
 - Automatically store session duration
 - Accurate daily and weekly totals

### Daily Report

 - Total study time for the day
 - Subject-wise breakdown
 - Most studied subject
 - Number of study sessions
 - Visual charts for clarity

### Weekly Report

 - 7-day study overview
 - Daily totals and weekly total
 - Average study time
 - Most productive day
 - Trend visualization

### Habit Tracker

 - Create custom daily habits
 - Mark habits as completed
 - Automatic streak tracking
 - Habit completion history

### Task Manager

 - Add tasks with subjects and due dates
 - Mark tasks as completed
 - View pending and due-today tasks
 - Integrated into the dashboard

### Leaderboards (Invite-Based)

### Settings

 - Add or remove subjects

 - Designed for future cloud sync

## Privacy First Design

This application is built with privacy as a core principle:

 - No screen recording
 - No keystroke logging
 - No access to personal content
 - Only user-approved data is shared
 - Leaderboards require explicit consent

Users always control their data.
